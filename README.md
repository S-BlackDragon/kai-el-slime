# Kai el Slime — build jugable

Roguelike de acción top-down en el navegador. Un slime contra El Pozo.

**Jugar:** https://s-blackdragon.github.io/kai-el-slime/

- **PC:** WASD mover · ratón apuntar · clic disparar · espacio/clic-der usable · shift dash · E comprar · R reiniciar
- **Móvil:** twin-stick táctil (palanca izquierda mover, derecha apuntar y disparar) + botones ⚡ dash / ◇ usable / € comprar. Mejor en horizontal.

Este repositorio contiene **solo el build compilado** del juego (en desarrollo).
El código fuente no está publicado.
